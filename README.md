# Payroll Tool (MVP)

## Tech Stack

- Frontend: Vue 3 + Vite + TypeScript + Naive UI
- Backend: Node.js + Express + TypeScript
- DB: (TBD / SQLite planned)

## Features (MVP)

- Employee Management (search)
- Attendance Management (search)
- Hourly Rate Management (search)
- Payroll Calculation (mock)
- Report Generation (mock)

## Run

```bash
npm run install:all
npm run dev
```
